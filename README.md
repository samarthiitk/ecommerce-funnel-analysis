# E-commerce Funnel Analysis (End-to-End Project)

## 📌 Overview
This project simulates user behavior in an e-commerce platform and performs end-to-end data analysis including funnel analysis, A/B testing, and SQL-based analytics.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- PostgreSQL
- SQL
- Git & GitHub

---

## 📊 Key Features
- Synthetic data generation (users, events, experiments)
- Data cleaning and merging (ETL pipeline)
- Funnel analysis (Python & SQL)
- Conversion rate analysis
- A/B testing with statistical validation (Z-test)
- Data visualization

---

## 📁 Project Structure
- `data.py` → Data generation & ETL pipeline
- `analysis_sql.py` → SQL-based analysis using PostgreSQL
- `analysis_python.py` → Python analysis, visualization & statistical testing

---

## 📈 Key Insights
- Significant drop-off observed from **view → add_to_cart**
- Variant A showed higher conversion than Variant B
- Statistical testing used to validate whether difference is significant

---

## 🚀 How to Run
1. Run `data.py` to generate data
2. Run `analysis_sql.py` for SQL analysis
3. Run `analysis_python.py` for Python analysis and visualization
